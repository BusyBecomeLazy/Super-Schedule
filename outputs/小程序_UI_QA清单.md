- 日程冲突卡片从数字角标堆叠改为扑克牌式层叠，顶层卡片保留完整标题和时间。

- 固定当前开发目录，并加入 `scripts/validate-miniprogram.js` 防止改错项目副本。
- 管理页群组表单统一为浅灰圆角输入框，角色选择增加箭头暗示。
- 管理页创建群组、加入群组、切换角色增加提交中状态和重复提交保护。
- 管理页切换开发账号增加确认弹窗。
- 日程页和课程页的自然语言输入区域统一为浅灰圆角表单样式。
- 日程页和课程页的 AI 录入从页面内嵌面板改为 Bottom Sheet，交互形态和新建日程/课程一致。
- AI 录入入口卡片改为和新增日程/课程相同的主操作样式。
- AI 录入解析结果改为确认面板样式，字段行、候选项、警告提示更易扫读。
- AI 录入文本变更时会清除旧解析结果，避免误确认过期结果。
- 日程卡片和课程卡片增加轻量按压反馈。
- 单个日程/课程卡片去掉阴影，重叠日程的扑克牌层叠继续保留阴影。
- 单个日程卡片去掉左侧粗边，避免被误认为还有重叠卡片。
- 无群组空状态增加“去管理页”主操作。
- 日程/课程网格空状态增加“新增日程/新增课程”主操作。
- 校验脚本增加 WXSS 花括号平衡检查。
- 校验脚本增加 `utils/config.ts` 和 `utils/config.js` 的 `LAN_HOST` 同步检查，避免真机预览连到旧地址。
- `package.json` 增加 `npm run validate:mp`，README 改为推荐用 npm 脚本运行小程序校验。
- 管理页完成视觉减法：开发账号切换弱化到右上角，权限网格改成标签，创建/加入群组拆成上下表单模块。
- 管理页群组和成员列表改成 Cell 布局，列表操作改为弱按钮/文字按钮，成员增加首字头像。
- 管理首页重构成类似微信“我/设置”的入口页，只保留个人信息卡片和“群组管理 / 成员权限设置 / 开发选项”三个入口。
- 管理页三个入口改为 Bottom Sheet 底部抽屉，不再跳转新页面；抽屉打开时隐藏自定义 TabBar。
